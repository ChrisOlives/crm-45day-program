### Main Project Task
**Build the GrowthBridge Automated Monthly Report System**

**Step 1: Build the Data Source**
Expand your Google Sheet "GrowthBridge_Client_Metrics":
- 30 rows — 3 months of data for 10 clients
- Clients: Apex Plumbing, Metro Dental, Sunrise Yoga, TechFix IT, Lakeview Homes, Bright Bites, Summit Law, Coastal Realty, FitStart Gym, Peak Accounting

**Step 2: Build the Report Generation Workflow**

Node 1 — Trigger: Schedule (1st of every month at 9:00 AM)

Node 2 — Google Sheets: Read all rows from "GrowthBridge_Client_Metrics"
Filter: Month = last month

Node 3 — Code node (JavaScript):
```javascript
// Group data by client name
const items = $input.all();
const clients = {};
for (const item of items) {
  const name = item.json['Client Name'];
  if (!clients[name]) clients[name] = [];
  clients[name].push(item.json);
}
// Return one item per client
return Object.keys(clients).map(name => ({
  json: {
    clientName: name,
    metrics: clients[name]
  }
}));
```

Node 4 — Loop Over Items (one execution per client)

Node 5 — Code node: Format the report HTML
Build an HTML email body with:
- Client name as heading
- Metrics table: Leads, Conversions, Revenue, Ad Spend, ROI (Revenue/Ad Spend)
- Month-over-month change (calculate % change vs. previous month)
- Simple "green arrow" or "red arrow" text indicator for each metric

Node 6 — Gmail: Send Email
- To: a fixed email (your own, simulating client email)
- Subject: "GrowthBridge Monthly Report — {{ $json.clientName }} — {{ $now.minus({months:1}).format('MMMM YYYY') }}"
- Body: the formatted HTML from Node 5

Node 7 — Google Sheets: Log sent reports
Sheet: "GrowthBridge_Report_Log"
Columns: Date Sent | Client Name | Report Month | Status | Email Subject
